# Roadmap

Our vision for democast. Help us build it.

---

## Phase 1: Core (Now ✅)

**Goal:** Functional web playground + API backend

- [x] Web playground (Next.js 15)
- [x] API backend (Express.js)
- [x] Mock render pipeline
- [x] Job queue system
- [x] Documentation
- [ ] Real ElevenLabs integration
- [ ] Real Playwright recording
- [ ] Real ffmpeg composition

**Timeline:** May 2026  
**Status:** 70% complete

---

## Phase 2: CLI (Q2 2026)

**Goal:** Command-line tool for developers

```bash
npm i -g democast
democast render demo.json
democast validate demo.json
democast list
```

- [ ] CLI package (`democast`)
- [ ] Config validation
- [ ] Error reporting
- [ ] Progress indicators
- [ ] Help documentation
- [ ] Shell completions (bash, zsh, fish)

**Timeline:** June 2026  
**Status:** Not started
